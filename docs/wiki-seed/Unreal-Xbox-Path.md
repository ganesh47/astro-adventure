# Unreal Xbox Path

Track what we learn about using Unreal Engine `5.7.4` for a Windows-first prototype and later Xbox Dev Mode or official Xbox release path.

## Current Direction

- Use Unreal Engine `5.7.4`.
- Use Windows desktop as the first validation target.
- Use a trusted self-hosted Windows runner for full Unreal validation.
- Keep Xbox deployment proof private-safe and defer it until M0 Unreal parity is stable.
