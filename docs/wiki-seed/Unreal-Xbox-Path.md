# Unreal Xbox Path

Track what we learn about using Unreal Engine `5.7.4` for a Windows-first prototype and later Xbox Dev Mode or official Xbox release path.

## Current Direction

- Use Unreal Engine `5.7.4`.
- Use Windows desktop as the first validation target.
- Use local/manual Windows builds for full Unreal validation unless a future scheduled runner policy is explicitly approved.
- Keep Xbox deployment proof private-safe and defer it until M0 Unreal parity is stable.

## Open Questions

- Exact local UE 5.7.4 install path, kept out of public docs and logs.
- Required Visual Studio and Windows SDK versions for repeatable packaging.
- Whether Xbox Dev Mode packaging can stay fully public-tooling based for M3.
- Which evidence belongs in private release records versus public status notes.
