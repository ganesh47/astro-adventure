# Asset Sources And Licensing

## Goal

Track public-safe asset sources and license decisions before anything enters the public repository.

## Candidate Sources

- NASA public media
- Kenney
- OpenGameArt
- Poly Haven
- ambientCG
- Original project-created assets
- AI-generated placeholders after review

## Review Rule

No asset should be committed to the public repo until it has an entry in `assets/manifest/assets.csv` with source, license, author, approval flags, and notes.
