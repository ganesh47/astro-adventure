# Astro Adventure v0.1.0-alpha.1

Release date: 2026-04-25

## Summary

This prerelease establishes Astro Adventure as an Unreal Engine 5.7.4 educational space game and adds the first source-only playable learning loop. The loop is designed for Windows PC validation first and keeps Xbox deployment work deferred to public-safe planning plus private evidence.

## Included

- Runtime Mercury, Mars, and Europa destination scene generated from C++ source.
- Controller and keyboard navigation for focus, scan/confirm, back, hint, and quiz answers.
- Discovery card text, one quiz per destination, gentle retry feedback, completion state, and local progress save.
- Age-banded lesson copy for ages 4-6, 7-9, and 10-12.
- Spaced repetition and mastery helper functions with Unreal automation tests.
- Public-safe research/spec docs for learning design, age bands, DevSecOps, progression, and Unreal/Xbox path.
- Public PR hygiene guardrails for generated files, private materials, asset manifest shape, and Unreal metadata.

## Validation Summary

Validated locally on Windows with Unreal Engine 5.7.4:

- Passed public hygiene: `powershell -NoProfile -ExecutionPolicy Bypass -File scripts\validate-public-repo.ps1`
- Passed Unreal editor compile with a local Unreal Engine 5.7.4 install.
- Passed Unreal Win64 game compile with a local Unreal Engine 5.7.4 install.
- Passed learning automation: `AstroAdventure.Learning` reported 2 succeeded, 0 failed.
- Passed command-line runtime smoke: `UnrealEditor-Cmd.exe ... /Engine/Maps/Entry -game -ExecCmds="Quit" -nullrhi`

## Known Limitations

- Alpha source release only; no public packaged Windows binary is attached.
- Runtime scene uses Unreal basic shapes and HUD text rather than final UMG art assets.
- Full package/cook validation is local/manual for now and is not run on public PRs.
- Xbox deployment, certificates, package artifacts, Partner Center material, and restricted SDK/GDK evidence stay out of the public repo.
